(PL)
1. Wchodzimy w plik "SteamAppIDs" i wpisujemy tam ID gier ze Steama, w których mają nabijać się godziny.
Kolejne numery wpisujemy po przecinku np.   730,710,240       i tak dalej.
2. Uruchamiamy plik "idler" i już. Aby zakończyć działanie programu, klikamy "ENTER"

Miłego idlowania!

(ENG)
1. Go to the "SteamAppIDs" file and enter the ID of the Steam games in which you want to add hours.
We enter subsequent numbers after a decimal point, e.g. 730,710,240 and so on.
2. Run the "idler" file and that's it. To end the program, click "ENTER"

Have fun idling!
